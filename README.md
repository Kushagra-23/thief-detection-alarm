# Theif Detection Alarm

Detect a person through webcam.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


### Technologies

1. Next JS
2. @tensorflow-models/coco-ssd
3. @tensorflow/tfjs
4. react-webcam
5. lodash
